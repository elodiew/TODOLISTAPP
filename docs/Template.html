show(data) → {string}

Source:
  • template.js, line 55

Crée un élément HTML et place le template dans l'application. Normalement, il est conseillé d'utiliser un moteur de template comme Mustache ou Handlebars. Pour simplifier notre application, nous avons décidé de créer un template en vanilla JS.

Example
view.show({
id: 1, // l'id de la tâche
title: "Hello World", // le titre de la tâche
completed: 0, // la tâche est par défaut non terminée
		});
Parameters:
Name Type Description
data object

L'objet contenant les informations à remplacer dans le template.
Returns:
