La fonction de rappel (seulement parce qu'on est en localStorage) utilisé après l'initialisation du Store. Dans le cas d'une vrai BDD, nous effecturions des appels AJAX.

Trouve un élément en fonction de la requête (objet JS)

Example
db.find({foo: 'bar', hello: 'world'}, function (data) {
	 // data will return any items that have foo: bar and
	 // hello: world in their properties
});
